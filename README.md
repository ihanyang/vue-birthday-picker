# vue-birthday-picker

> 移动端版日期选择组件 [demo](http://sw2016.h5.88h5.cn/profile)

# Installation

## npm
`$ npm install vue-birthday-picker`

# Usage
```vue
<template>
	<date-picker :status.sync="status" :year-value.sync="year" :month-value.sync="month" v-if="status"></date-picker>
</template>
```
